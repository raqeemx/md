# محرر ماركداون | Markdown Editor

محرر ملفات Markdown يعمل بالكامل في المتصفح (Static Web App) مع **معاينة مباشرة** ودعم ممتاز للغة **العربية والإنجليزية** واتجاهي النص **RTL / LTR**.

## 🎯 فكرة المشروع

تطبيق ويب خفيف وسريع يتيح فتح ملفات `.md` من الجهاز، تعديلها في محرر نصوص، ومشاهدة النتيجة المنسّقة فوراً أثناء الكتابة، ثم حفظ الملف مرة أخرى بصيغة `.md` — كل ذلك دون أي خادم (Server-less)، فجميع العمليات تتم على جهاز المستخدم.

## ✅ الميزات المكتملة

- **فتح الملفات**: زر "فتح" + دعم السحب والإفلات (Drag & Drop) لملفات `.md` / `.markdown` / `.txt`
- **محرر نصوص** بخط Monospace مع دعم Tab لإدراج مسافات
- **معاينة مباشرة (Live Preview)** مع Debounce لتحسين الأداء
- **دعم كامل لتنسيقات Markdown** (GitHub Flavored):
  - العناوين، القوائم (مرتبة/نقطية/مهام)، الروابط، الصور
  - الجداول، الأكواد مع تمييز لوني (highlight.js)، الاقتباسات، الخطوط الفاصلة
- **دعم العربية والإنجليزية**:
  - واجهة مترجمة بالكامل (زر تبديل اللغة)
  - كل سطر في المحرر والمعاينة يأخذ اتجاهه تلقائياً (`unicode-bidi: plaintext`)
  - زر تبديل يدوي بين RTL و LTR
  - الأكواد تبقى دائماً LTR حتى داخل واجهة RTL
- **حفظ الملف** بصيغة `.md` بترميز UTF-8 (زر حفظ + اختصار Ctrl/Cmd+S)
- **زر تفريغ المحتوى** مع رسالة تأكيد
- **3 أوضاع عرض**: محرر فقط / مقسوم / معاينة فقط
- **الوضع الليلي والنهاري** مع تبديل ثيم تمييز الأكواد تلقائياً
- **شريط حالة**: عدد الكلمات والأحرف والأسطر + مؤشر التعديلات غير المحفوظة
- **إشعارات Toast** + تحذير قبل مغادرة الصفحة مع تعديلات غير محفوظة
- **حفظ التفضيلات** (اللغة/الاتجاه/المظهر/وضع العرض) في localStorage
- **تصميم متجاوب** بالكامل مع الهاتف والكمبيوتر

## 📂 هيكل الملفات

```
index.html          الصفحة الرئيسية (الهيكل + شريط الأدوات + المحرر والمعاينة)
css/
  └── style.css     التصميم: متغيرات الثيم، RTL/LTR، الوضع الليلي، التجاوب
js/
  └── main.js       المنطق: i18n، الملفات، المعاينة، الثيم، الاختصارات
README.md           هذا الملف
```

## ⌨️ اختصارات لوحة المفاتيح

| الاختصار | الوظيفة |
|----------|---------|
| `Ctrl/Cmd + S` | حفظ / تحميل الملف |
| `Ctrl/Cmd + O` | فتح ملف |
| `Tab` (داخل المحرر) | إدراج 4 مسافات |

## 🛠️ التقنيات والمكتبات

- HTML5 / CSS3 / Vanilla JavaScript (ES6+)
- [marked.js](https://marked.js.org) — تحويل Markdown إلى HTML
- [DOMPurify](https://github.com/cure53/DOMPurify) — تعقيم HTML لمنع XSS
- [highlight.js](https://highlightjs.org) — تمييز لوني للأكواد
- Font Awesome + خطوط IBM Plex Sans Arabic / Inter / JetBrains Mono
- لا يوجد تخزين خادمي — كل البيانات على جهاز المستخدم (FileReader + Blob + localStorage للتفضيلات)

## 🚀 طريقة التشغيل

1. افتح `index.html` مباشرة في أي متصفح حديث، أو انشر المشروع من تبويب **Publish**.
2. اضغط "فتح" لاختيار ملف `.md` أو اسحبه وأفلته في النافذة.
3. عدّل النص في المحرر وشاهد المعاينة الفورية.
4. اضغط "حفظ" لتحميل الملف المعدّل بصيغة `.md`.

## 🔜 ميزات غير منفّذة بعد

- شريط أدوات تنسيق (أزرار Bold/Italic/إدراج جدول...)
- تصدير إلى PDF أو HTML
- التمرير المتزامن بين المحرر والمعاينة

## 💡 اقتراحات للتطوير المستقبلي

1. **شريط أدوات تنسيق** لإدراج صيغ Markdown بنقرة واحدة
2. **تصدير PDF/HTML** عبر `window.print()` أو مكتبة html2pdf
3. **التمرير المتزامن (Synced Scrolling)** بين اللوحين
4. **حفظ تلقائي** للمسودة في localStorage واستعادتها عند العودة
5. **دعم File System Access API** للحفظ المباشر فوق الملف الأصلي (Chrome/Edge)
6. **إدارة ملفات متعددة** بتبويبات + تخزين عبر RESTful Table API
7. **بحث واستبدال** داخل المحرر
8. **دعم Mermaid** للرسوم البيانية و KaTeX للمعادلات الرياضية
